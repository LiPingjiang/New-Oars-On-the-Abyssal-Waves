## 案例收录标准提醒

每个案例必须满足：
1. 有明确的受雇经历
2. 主动脱离雇佣关系
3. 从零建立稳定自有事业
4. 史料/资料可考证

**排除**：富二代继承、天生商人、世袭经营、无打工经历直接创业、短期摆摊
