## 案例收录标准提醒

每个案例必须满足：
1. 有明确的受雇经历
2. 主动脱离雇佣关系
3. 从零建立稳定自有事业
4. 史料/资料可考证（现代案例要求多源交叉验证）

**排除**：富二代继承、天生商人、世袭经营、无打工经历直接创业、短期摆摊

## 现代案例写作规范

1. **史料来源与交叉验证**：每个现代案例必须列出具体的信息来源（官方资料、SEC文件、权威媒体、传记等），并说明多源交叉验证情况
2. **可信度评估**：对每个案例的关键事实进行可信度评估（高/中/低），并说明理由
3. **研究边界**：明确标注案例聚焦的"转型阶段"，后期发展简略带过
4. **跨案例对照**：每个案例小结需包含与其他案例的对比表格，确保跨时代可比性
