# 核心问题与研究边界

## 一、核心研究问题

不同时代、不同文明下，拥有雇员身份的普通人，依靠哪些**个人条件、外部机遇、行动策略**，突破雇佣束缚，完成从"为他人劳作"到"拥有自有事业"的身份跃迁？

## 二、研究对象硬性筛选标准（缺一不可）

### 必须同时满足

1. **前期身份**：长期依附他人、领取报酬的雇佣劳动者
   - 古代：雇工、伙计、学徒、衙门小吏、长工、短工
   - 近代：工厂工人、洋行职员、店铺学徒
   - 现代：企业上班族、打工人、基层职员、合同工

2. **关键行为**：主动脱离雇佣关系，从零搭建属于自己的经营主体
   - 小作坊、商铺、商行
   - 实业公司、工厂
   - 个人品牌产业、工作室
   - 必须是稳定、持续的经营实体，非一次性投机

3. **研究截取阶段**：只聚焦转型全过程
   - 打工时期的积累
   - 触发辞职的契机
   - 起步阶段的困境
   - 从零建立自有事业
   - 后期做大成集团：简略带过，不作为重点

### 直接排除人群

| 排除类型 | 说明 |
|----------|------|
| 富二代接手家族生意 | 无打工经历，直接继承 |
| 天生商人 | 自幼经商，无受雇阶段 |
| 世袭经营者 | 家族世代经商，本人未经历雇佣 |
| 无打工经历直接创业 | 毕业即创业、从未受雇 |
| 单纯摆摊短期倒卖 | 未形成稳定自有事业，属临时谋生 |
| 被迫转行（非主动） | 如被裁员后被动失业，非主动选择创业 |

## 三、学科定位

### 三层归属（主次清晰）

1. **核心基底：经济社会史**
   - 最贴合史学属性
   - 记录不同时代打工人如何跳出雇佣体系
   - 关注制度、经济结构对个体转型的约束与机遇

2. **中层支撑：职业社会学、身份流动研究**
   - 专门研究"打工人→老板"阶层跨越
   - 关注社会资本、文化资本、代际流动

3. **辅助工具：初创创业理论**
   - 只用到离职、起步、零资源起家的前置阶段
   - 不用复杂集团管理理论、战略管理理论

### 与相关学科切割

**区别传统商业史**
- 传统商业史：多写大商人、世家、行业兴衰、宏观经济
- 本研究：只抓有过打工/受雇经历的人，剔除生来经商、世袭家业者
- 视角：从底层向上看，而非从资本向下看

**区别商学院创业学**
- 商学院创业学：面向当下商业实操，侧重可复制的商业模式、融资、管理
- 本研究：底色是史学对比，古今中外案例对照，挖掘转型的共性、时代差异
- 目标：理解"为什么能转型"而非"如何创业成功"

**区别人物传记**
- 人物传记：讲故事、塑形象、渲染个人魅力
- 本研究：拆解「雇佣身份→自主事业」这一段转变全过程，用统一分析框架量化比较

## 四、研究核心限定（防止越界）

### 聚焦范围

- **只研究"转型阶段"**：从决定脱离雇佣到自有事业初步稳定
- **不深挖后期扩张**：企业做大后的治理、并购、多元化简单略写
- **区分副业与转型**：短期副业增收不算研究核心，重点是主动脱离全职雇佣，把自主事业作为核心生存依托
- **统一标尺**：无论古代伙计还是现代白领，都用同一套维度分析

### 时间跨度

- **前近代**：古代至1840年（鸦片战争前）
- **近代工业化**：1840-1949年
- **现当代**：1949年至今

## 五、研究价值

1. **填补空白**：现有史料要么写大企业家，要么单纯讲现代职场，很少专门聚焦"有打工经历的普通人转型"做贯通古今的史学对比
2. **视角独特**：完全站在底层雇员个人视角，而非资本、企业、行业宏观视角，侧重普通人的自我突破
3. **实用价值**：古代学徒、近代工人、现代白领跨越千年的相似困境与解法，能给当下朝九晚五想创业的人提供跨越时代的参考，不局限于当下短视频、商科碎片化经验
