# 领域划分说明

## 核心逻辑

领域划分是研究框架的**中层分类维度**，按"打工期间积累的核心资本类型"将研究对象分为四大领域。技术工匠靠手艺积累，商业伙计靠客户资源积累，知识吏员靠信息认知积累，服务者靠人脉网络积累——古今同一工作本质的案例归入同一领域，便于提炼跨时代规律。

每个领域下设有**案例库**，案例库是领域下的具体研究对象，是研究体系的叶子节点。

---

## 领域一：技术-工匠型（核心积累：手艺、技术、工艺知识）

> 古代学徒在作坊学手艺，现代工程师在实验室练技术——本质都是**以技术技能为核心资本**，转型路径是"技术变现"或"以技术差异化切入市场"。

### 已收录案例

- **黄仁勋：从AMD/LSI芯片工程师到NVIDIA创始人**（美国硅谷·1993年）——已完成五维度分析

### 待收录案例方向

- **现当代**：早期硅谷工程师创业（如AMD/Intel早期员工创业）、中国硬件工程师转型、开源社区技术创业、生物/医药技术员创业
- **近代工业化**：工厂技术工人→独立工厂主（如无锡荣氏兄弟从钱庄学徒到纺织业？需验证打工经历）、江南制造局技术工人自立
- **前近代**：景德镇陶瓷工匠自立窑口、徽州墨工独立开墨坊、江南织工独立设机、木匠/铁匠自立作坊

## 领域二：商业-贸易型（核心积累：客户资源、货源渠道、商业信息）

> 古代盐店伙计掌握主顾关系和货源信息，近代洋行买办熟悉进出口流程，现代销售掌握客户名单——本质都是**以商业信息和客户关系为核心资本**，转型路径是"依托原资源自立门户"或"信息套利"。

### 已收录案例

- **程白庵：从盐店伙计到独立盐商**（明代·徽州）——已完成五维度分析

### 待收录案例方向

- **现当代**：BAT早期销售离职创业、跨境电商从业者自立（如亚马逊卖家、SHEIN早期员工）、平台经济个体户（淘宝/拼多多店主）、房地产经纪人自立中介公司
- **近代工业化**：洋行买办自立（唐廷枢、徐润从洋行到轮船招商局/独立矿业）、晋商/徽商伙计→掌柜→东家、广州十三行伙计自立
- **前近代**：明清徽商/晋商伙计→掌柜→东家、广州十三行伙计自立、漕帮成员转型独立运输商

## 领域三：知识-信息型（核心积累：信息处理、系统认知、管理能力）

> 古代衙门胥吏熟悉政务流程和人情网络，现代产品经理掌握用户行为和商业模式——本质都是**以系统认知和信息处理能力为核心资本**，转型路径是"模式创新"或"认知套利"。

### 已收录案例

- **埃隆·马斯克：从Zip2/PayPal到SpaceX/Tesla连续创业者**（美国·1995-2002年）——已完成五维度分析

### 待收录案例方向

- **现当代**：咨询公司顾问自立（如麦肯锡/贝恩离职创业）、数据分析师创业、产品经理做SaaS、连续创业者的模式复制、程序员做工具类产品
- **近代工业化**：银行职员→独立金融家、报社编辑→独立出版人、洋行文员→独立贸易商
- **前近代**：衙门胥吏→幕僚/商人、书院塾师→独立学派、账房先生→独立钱庄

## 案例收录标准提醒

每个案例必须满足：
1. 有明确的受雇经历
2. 主动脱离雇佣关系
3. 从零建立稳定自有事业
4. 史料/资料可考证（现代案例要求多源交叉验证）

**排除**：富二代继承、天生商人、世袭经营、无打工经历直接创业、短期摆摊

## 现代案例写作规范

1. **史料来源与交叉验证**：每个现代案例必须列出具体的信息来源（官方资料、SEC文件、权威媒体、传记等），并说明多源交叉验证情况
2. **可信度评估**：对每个案例的关键事实进行可信度评估（高/中/低），并说明理由
3. **研究边界**：明确标注案例聚焦的"转型阶段"，后期发展简略带过
4. **跨案例对照**：每个案例小结需包含与其他案例的对比表格，确保跨类型可比性
