# 术语表

## 核心概念

| 术语 | 释义 |
|------|------|
| 执役 | 手持、侍奉、听命于他人的雇佣劳动者，包括古代伙计、近代工人、现代上班族 |
| 开疆 | 开辟属于自己的事业版图、产业疆域，区别于单纯摆摊糊口 |
| 改楫 | 改换手中船桨，喻指从依附他人的谋生工具转向自主掌控的人生航向 |
| 溟波 | 幽深无际的沧海浪潮，喻指千年谋生的宏大环境与职场风浪 |
| 雇佣劳动者 | 长期依附他人、领取报酬的劳动者，区别于自雇者、业主 |
| 转型 | 从雇佣身份向自主事业所有者身份的转变过程 |
| 身份跃迁 | 社会经济地位从受雇者向业主阶层的跨越 |

## 古代术语

| 术语 | 释义 |
|------|------|
| 学徒 | 在手工业作坊或商铺中学习技艺和经营的年轻人，通常三年为期 |
| 伙计 | 商铺中的受雇职员，介于学徒与掌柜之间 |
| 掌柜 | 商铺的管理者，通常由资深伙计升任 |
| 东家 | 商铺或企业的所有者 |
| 行会 | 同行业手工业者或商人的组织，控制准入、定价、技术标准 |
| 买办 | 近代洋行中受雇的中国籍高级管理人员，掌握国际贸易知识 |
| 跑街 | 洋行或商铺中负责外勤、招揽业务的职员 |

## 近代术语

| 术语 | 释义 |
|------|------|
| 洋行 | 近代外国资本在中国开设的商行 |
| 实业 | 相对于商业贸易，指制造业、工业等实体产业 |
| 商会 | 商人自发组织的利益代表机构 |
| 下海 | 1980年代机关干部、国企员工辞职经商的俗称 |

## 现代术语

| 术语 | 释义 |
|------|------|
| 白领 | 企业中的非体力劳动职员 |
| 上班族 | 受雇于企业、按固定时间工作的职员 |
| 副业 | 主业之外的兼职收入来源 |
| 平台经济 | 基于互联网平台的新型经济形态 |
| 天使投资 | 初创企业早期阶段的风险投资 |
| 众筹 | 通过互联网向大众募集小额资金 |
