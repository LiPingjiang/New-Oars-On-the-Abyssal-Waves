## 核心命题

不同时代、不同文明下，拥有雇员身份的普通人，依靠哪些**个人条件、外部机遇、行动策略**，突破雇佣束缚，完成从"为他人劳作"到"拥有自有事业"的身份跃迁？

从古代商铺伙计自立门户，到近代工厂工人转型实业家，再到现代白领离职创业——跨越千年的"打工人突围"故事，隐藏着怎样的共性规律与时代差异？

## 研究边界（硬性筛选标准）

研究对象必须同时满足：

1. **前期身份**：长期依附他人、领取报酬的雇佣劳动者（古代雇工/伙计/学徒、近代工厂工人/洋行职员、现代上班族/基层职员）
2. **关键行为**：主动脱离雇佣关系，从零搭建属于自己的经营主体（小作坊、商铺、商行、实业公司、个人品牌产业均可）
3. **研究截取阶段**：只聚焦转型全过程——打工时期的积累、触发辞职的契机、起步阶段的困境、从零建立自有事业；后期做大成集团简略带过

**直接排除**：富二代接手家族生意、天生商人、世袭经营者、无打工经历直接创业的人、单纯摆摊短期倒卖（未形成稳定自有事业）

---

## 统一分析维度（所有案例通用框架）

每个案例均按以下五维度拆解，确保跨时代可比性：

1. **打工阶段的底层积累**：技能、人脉、微薄资本、行业认知
2. **转型触发点**：内卷、薪资瓶颈、行业风口、政策松动、失业、个人抱负
3. **转型阻碍**：资金匮乏、原有雇主打压、社会制度限制、家庭压力、缺乏资源
4. **破局手段**：合伙、小额借贷、副业过渡、差异化小众赛道、依托原有打工资源创业
5. **时代制度影响**：重农抑商、行会垄断、工商政策、产权保护程度如何左右转型难度

## 研究特色

- **填补空白**：现有史料要么写大企业家，要么讲现代职场，很少专门聚焦"有打工经历的普通人转型"做贯通古今的史学对比
- **视角独特**：完全站在底层雇员个人视角，而非资本、企业、行业宏观视角，侧重普通人的自我突破
- **实用价值**：古代学徒、近代工人、现代白领跨越千年的相似困境与解法，给当下朝九晚五想创业的人提供超越碎片经验的参考
- **从后往前**：现代资料更全、更易交叉验证，先写现代建立标准，再逐步向前推进
- **按工作本质分类**：不按时代分段，而按打工积累的核心资本类型划分，古今同类工作归为一组，更易提炼跨时代规律

## 书名释义

**溟波改楫**：众生沉浮于千年谋生沧海浪潮之中，不甘终生持他人之桨随波逐流，决意改换手中船楫，自主掌控前路航程。

**从执役到开疆**：研究一切由受人驱使的雇佣者，转身开拓自有产业疆域的全过程。
