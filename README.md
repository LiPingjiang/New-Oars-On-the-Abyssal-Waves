# 溟波改楫：从执役到开疆

> 跨时代雇佣劳动者自主创业转型历史比较研究

---

## 项目简介

本研究聚焦于一个被长期忽视的历史命题：**拥有雇员身份的普通人，如何在不同时代、不同文明下，突破雇佣束缚，完成从"为他人劳作"到"拥有自有事业"的身份跃迁？**

从古代商铺伙计自立门户，到近代工厂工人转型实业家，再到现代白领离职创业——跨越千年的"打工人突围"故事，隐藏着怎样的共性规律与时代差异？

## 核心研究问题

不同时代、不同文明下，拥有雇员身份的普通人，依靠哪些**个人条件、外部机遇、行动策略**，突破雇佣束缚，完成身份跃迁？

## 研究特色

- **填补空白**：现有史料要么写大企业家，要么讲现代职场，很少专门聚焦"有打工经历的普通人转型"做贯通古今的史学对比
- **视角独特**：完全站在底层雇员个人视角，而非资本、企业、行业宏观视角
- **实用价值**：跨越千年的相似困境与解法，给当下朝九晚五想创业的人提供超越碎片经验的参考

## 项目结构

| 目录 | 内容 |
|------|------|
| `研究框架/` | 核心方法论、研究边界、分析维度 |
| `案例库/` | 按时代分段的转型案例（前近代/近代工业化/现当代） |
| `比较研究/` | 跨时代对比分析、共性规律、失败诱因 |
| `附录/` | 术语表、参考文献 |

## 研究边界

**研究对象硬性标准**（缺一不可）：
1. 前期身份：长期依附他人的雇佣劳动者
2. 关键行为：主动脱离雇佣关系，从零搭建自有经营主体
3. 研究截取：只聚焦转型全过程，后期做大成集团简略带过

**直接排除**：富二代接手家族生意、天生商人、世袭经营者、无打工经历直接创业的人

## 统一分析维度

所有案例均按以下五维度拆解：
1. 打工阶段的底层积累
2. 转型触发点
3. 转型阻碍
4. 破局手段
5. 时代制度影响

## 书名释义

**溟波改楫**：众生沉浮于千年谋生沧海浪潮之中，不甘终生持他人之桨随波逐流，决意改换手中船楫，自主掌控前路航程。

**从执役到开疆**：研究一切由受人驱使的雇佣者，转身开拓自有产业疆域的全过程。
